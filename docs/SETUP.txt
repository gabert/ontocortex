SETUP — QUICK REFERENCE
=======================

This file is a quick reference. For the full guide, see docs/USER_GUIDE.md.


1. INSTALL
----------
    git clone https://github.com/gabert/ontocortex.git
    cd ontocortex
    pip install -r requirements.txt
    docker-compose up -d


2. CONFIGURE
------------
    cp config.ini.template config.ini

Edit config.ini:
  - Set your Anthropic API key (or use ANTHROPIC_API_KEY env var)
  - Set LLM models in [models]
  - Set chat/architect parameters in [chat] and [architect]
  - Add a [data_source:NAME] section for each domain database

All sections and all keys are required — there are no defaults.
See config.ini.template for the annotated reference.


3. GENERATE SCHEMA (one-time per domain, no LLM needed)
--------------------------------------------------------
    python scripts/design_plan.py student_loans
    python scripts/build_schema.py student_loans
    # Optional: generate demo seed data (requires LLM + API key)
    python scripts/build_schema.py student_loans --seed


4. RUN
------
    python main.py student_loans           # CLI agent
    streamlit run streamlit_app.py         # Web UI (http://localhost:8501)

The database is created automatically on first launch.


5. ADD A NEW DOMAIN
-------------------
See "Creating a new domain" in docs/USER_GUIDE.md.
